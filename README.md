Shopify App Server
=====
+ Firebase
-----

### Local Setup

1. (sudo) npm install
2. npm run watch
3. In a new terminal tab:
..* npm start
4. Server will now be running on port 3030
5. Navigate to 127.0.0.1:3030 to view page
6. Create a new branch.
7. Beging editing in src folder.
